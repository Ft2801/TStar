# Inspiration

TStar has been heavily inspired by and has drawn some of its functionalities, algorithms, and workflows from other incredible astrophotography software. We would like to express our gratitude and acknowledge the following projects:

## Siril
[Siril](https://siril.org/) is an astronomical image processing tool created to process images. Some of the features and processing methodologies in TStar have been implemented taking inspiration from the workflow, capabilities, and the work of the Siril team. 
*Important License Note:* Siril is distributed under the **GNU General Public License v3.0 (GPLv3)**. Because TStar incorporates code from Siril (e.g., in the `src/stacking` module), a copy of this license has been included in the repository as `LICENSE-GPL`. Anyone distributing TStar binaries that include this code must comply with the GPLv3 terms.

## Seti Astro Suite
[Seti Astro Suite](https://www.setiastro.com/) provides various tools for astrophotography. Certain contents and scripts within TStar have been inspired by Seti Astro Suite.

We thank these teams for their immense contributions to the astronomy and astrophotography community!
